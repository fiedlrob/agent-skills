# Agent Skills

Agent Skills is a public collection of reusable skills for AI coding agents.

The repository is intended to keep agent-facing workflows, conventions, and
task-specific instructions versioned as plain text. Skills should be easy to
review, copy, adapt, and improve through pull requests.

## Goals

- Maintain practical skills for recurring AI-agent tasks.
- Keep skill instructions transparent and reviewable.
- Provide a stable place for experiments that can later be promoted into more
  general agent workflows.
- Encourage contributions through issues and pull requests.
