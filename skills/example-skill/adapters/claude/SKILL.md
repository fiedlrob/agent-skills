# Claude Adapter

Read the tool-agnostic core instructions in `../../SKILL.md` before acting.

When adapting this template for Claude:

1. Keep `name` lowercase with hyphen-separated words.
2. Rewrite `description` so it clearly states when Claude should use the skill.
3. Keep detailed workflow, safety notes, references, and scripts in the core
   skill directory unless Claude needs a tool-specific override.
